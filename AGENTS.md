# AGENTS.md

## Purpose of this repository

This repository is a lightweight sandbox for:

- checking how the assistant works in web agent mode
- asking generic software and programming questions
- optionally trying small code or workflow tasks

## How to use this repo with the assistant

You can use this repo in two ways:

1. **Q&A mode**: ask conceptual questions about software, architecture, debugging, languages, tooling, and best practices.
2. **Hands-on mode**: ask for file edits, scripts, tests, refactors, or git operations inside this repository.

## Conversation memory / summaries

To keep lightweight continuity across chats, store short session summaries in:

- `CONVERSATION_SUMMARIES.md`

Suggested entry format:

- Date
- User request(s)
- Key actions taken
- Result / status
- Next steps (if any)

## Notes

- Keep summaries short and practical.
- Avoid storing sensitive credentials or personal data in summaries.
